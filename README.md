# WORM (Will's ORM)

## Overview

WORM is a SQL-first ORM built for PostgreSQL databases for typescript, with inspiration taken from [sqlc](https://sqlc.dev) and [Drizzle ORM](https://orm.drizzle.team).

The goal is to have an ORM that lets me define my database schema and queries in raw SQL without giving up type-safety.

This project was created primarily for learning purposes and is therefore highly unlikely to ever be something you would want to use in production, though feedback and contributions are always welcome.
